# PRB.111.075426_Ice_Growth_Sphere

This repository contains the data used for the figures within the project "Mechanism for ice growth on the surface of a spherical water droplet."
The data can be found within the file "data.xls"
The final paper was published on Febuary 21st 2025 in Physical Review B. (https://journals.aps.org/prb/abstract/10.1103/PhysRevB.111.075426)
A preprint vesion can be accessed here: https://arxiv.org/abs/2502.13155
